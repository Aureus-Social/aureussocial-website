# aureussocial-website

Site vitrine commercial d'**Aureus Social Pro** — `aureussocial.be`

## Pages
- `index.html` — Landing principale (hero, 12 fonctionnalités, module Optimisation, comparaison, PME/fiduciaires, tarifs, FAQ)
- `optimisation.html` — Page produit dédiée Module Optimisation (10 leviers détaillés)

## Stack
HTML / CSS / JavaScript vanilla — aucune dépendance.
Fonts via Google Fonts CDN : **Fraunces** (titres) + **Inter** (body).

## Déploiement local
Aucun build. Ouvrir `index.html` directement dans le navigateur.

## Déploiement production
Site statique. Compatible Vercel / Netlify / GitHub Pages.

Domaine cible : `aureussocial.be` (à configurer côté DNS + plateforme).
